## Description

minitalk allows communication across processes via ther pid (process id).
We do this by sending individual bits (01) across processes, this may come at anytime during the execution of code, even in between a "line" of code. This project aims to teach about deadlocks and race conditions and concurrent coding.

## Instructions

1. Run `make`
2. Run `./server`, this will give you a pid
3. Run `./client <pid> <message>`
